## TST002 Login de Usuário na Plataforma

**Caso de uso que se baseia:** CDU002 Fazer login

**Cenário:** Fluxo de Login de Usuário com Credenciais Válidas

**Requisitos associados:**

* RF001: O sistema deve validar as credenciais de login (e-mail e senha).
* RF002: O sistema deve conceder acesso ao usuário autenticado.
* RF003: O sistema deve direcionar o usuário à tela inicial da plataforma após login bem-sucedido.

**Preparação:**

a) O estudante ou empreendedor deve possuir um cadastro ativo na plataforma TechStart com as seguintes credenciais:

| Nome/Razão Social     | CPF/CNPJ       | E-mail                   | Senha       | Empresa/Estudante |
| --------------------- | -------------- | ------------------------ | ----------- | ----------------- |
| Enzo Henry da Rosa    | 59701806026    | enzo.rosa@email.com.br   | enzo1234    | Estudante         |
| Empresa de Teste Ltda | 83416264000158 | empresa@testeltda.com.br | empresa1234 | Empresa           |

b) O sistema deve estar em funcionamento e acessível através do navegador.

**Passos para execução do teste:**

1. Acessar a página inicial do sistema TechStart
2. Clicar no botão de “Login”
3. Informar e-mail e senha válidos
4. Clicar no botão “Entrar”

**Resultado esperado:**

a) O sistema valida as credenciais e exibe a mensagem:

> **"Login realizado com sucesso"**

b) O usuário é direcionado para a tela inicial da plataforma, com acesso liberado às funcionalidades correspondentes ao seu perfil (Estudante ou Empreendedor).

**Cenário Alternativo:** Recuperação de Senha

**Requisitos associados:**

* RF004: O sistema deve permitir a recuperação de senha através de e-mail cadastrado.
* RF005: O sistema deve enviar instruções de redefinição de senha por e-mail.
* RF006: O sistema deve permitir redefinição e posterior login com a nova senha.

**Preparação:**
O usuário deve possuir um e-mail cadastrado na plataforma.

**Passos para execução do teste:**

1. Acessar a página de login da plataforma
2. Clicar no link “Esqueci a senha”
3. Informar o e-mail cadastrado
4. Verificar o recebimento do e-mail com instruções de redefinição
5. Seguir o link enviado e cadastrar uma nova senha
6. Retornar à tela de login e realizar o acesso com a nova senha

**Resultado esperado:**

a) O sistema envia o e-mail com o assunto:

> **"Redefinição de senha - TechStart"**

b) Após redefinir, o usuário deve conseguir efetuar login normalmente com a nova senha.

**Cenário de Exceção:** Credenciais Inválidas

**Requisitos associados:**

* RF007: O sistema deve validar e identificar credenciais incorretas.
* RF008: O sistema deve exibir mensagem de erro quando o login falhar.

**Preparação:**
Utilizar um e-mail ou senha incorretos.

**Passos para execução do teste:**

1. Acessar a página de login da plataforma
2. Informar e-mail e/ou senha inválidos
3. Clicar no botão “Entrar”

**Resultado esperado:**

a) O sistema exibe a mensagem de erro:

> **"E-mail ou senha incorretos. Tente novamente."**

b) O sistema não permite o acesso e mantém o usuário na tela de login.
